Categorie : Formation

Site internet crée pour le quatrième projets de ma formation développeur web.
je devais créer un site internet pour un écrivain qui souhaitais publier son nouveau roman par épisode sur son site internet.
Le site devais utiliser le php / MySQL car il ne souhaitais pas utiliser de CMS style wordpress il voulais son propre outil de blog.
Le site devais répondre à certain critère, il devais respecter une architecture MVC, avoir une interface WYSIWYG basée sur TinyMCE pour la rédaction des articles.
